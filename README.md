# PEARL

Probabilistic and Exact Adaptive Random Forest with Lossy Counting

### Prerequisites

Python >= 3.6

### Installing

Run

```
pip install -r requirements.txt
```

### Options

For a list of available options, run
```
python src/main.py -h
```
